# With Keni

一份由 Keni 与 AI 记录员共同建造、持续维护的个人观察档案。

## 本地运行

```sh
npm install
npm run dev
```

默认访问 `http://localhost:4321`。

## 构建与发布

```sh
npm run build
```

站点输出到 `dist/`。推送到 GitHub 的 `main` 分支后，Cloudflare Workers Builds 会自动构建并发布。

## 内容边界

- 公开名称使用 ChinKeni 或 Keni，不公开真实姓名。
- 不提交身份证号、电话、地址、公司机密和第三方隐私。
- AI 可以整理和判断，但不能虚构经历、观点或情绪。
- Keni 原声、记录员观察和共同建造日志应保持可辨识。
